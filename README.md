================================
THE SYNTAX ESCAPE: SYSTEM.REBOOT
================================
------------------------------------------------------------------------------------------------------------------------------------------
- The visual style is Cyber-Minimalist with a "Dark Mode" aesthetic. The environment is set inside a virtual circuit board, featuring neon green or blue lines against a deep black background. The main character is a pixelated sprite representing a cursor or a small robot. The overall atmosphere is high-tech and slightly tense, using "glitch" visual effects and terminal-style fonts to make the player feel like they are truly inside a computer.
